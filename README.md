# Bank Management System (Java)

## Project Overview

This project is a **console-based Bank Management System** developed using **Core Java**.
It simulates basic banking operations like creating accounts, depositing money, withdrawing money, and checking balances.

The goal of this project was to understand and practice **Object-Oriented Programming concepts in Java**, along with using **ArrayList, methods, and conditional logic**.

This project was created as part of a **Java weekly learning project** to strengthen practical programming skills.

---

## Features

The system allows a user to perform the following operations:

* Create a new bank account
* Deposit money into an account
* Withdraw money from an account
* Check account balance
* View all existing accounts
* Transfer money between accounts
* Delete an account
* View transaction history
* PIN-based security for account access

Each account is protected with a **4-digit PIN**, which adds a basic layer of security before performing sensitive operations.

## How the System Works

1. The program runs in the **console** and displays a menu with different banking options.
2. The user selects an option by entering a number.
3. Based on the choice, the program performs the corresponding operation.
4. All accounts are stored temporarily using an **ArrayList** during the program execution.

Since this is a console-based application, the data is **not stored permanently** and will reset when the program stops.
